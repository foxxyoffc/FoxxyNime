# Anime Streaming Miniapp with VIP Dracin

## Setup Instructions

### Environment Variables (Backend)
- `MONGO_URI` - MongoDB Atlas connection string
- `BOT_TOKEN` - Telegram bot token from @BotFather
- `WEBHOOK_URL` - Your backend URL (e.g., https://your-app.vercel.app)
- `FRONTEND_URL` - Your frontend URL (e.g., https://frontend.vercel.app)
- `PORT` - 3000

### Deploy
1. Push this repo to GitHub
2. Import to Vercel (backend folder)
